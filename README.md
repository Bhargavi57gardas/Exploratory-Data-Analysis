# 📊 Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on a dataset using Python. It includes data cleaning, descriptive statistics, trend analysis, outlier detection, and data visualization to uncover meaningful insights.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
